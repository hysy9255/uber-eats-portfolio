
# 우버 잇츠 어플리케이션

## 개요 
음식 배달 서비스의 흐름을 구현한 풀스택 애플리케이션으로, 사용자 역할 기반 기능과 실시간 주문 처리 기능을 포함하고 있습니다.

## 기술 스택 
* Node.js
* NestJS
* PostgreSQL
* TypeORM 
* GraphQL
* Swagger
